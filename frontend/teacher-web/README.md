# Teacher Web (教师端)

管理后台 Web 应用，用于教师查看学生测评结果、生成入口链接、分享报告给家长。

## 技术栈 (推荐)

- **Framework**: React / Vue 3
- **UI Library**: Ant Design / Element Plus
- **Build Tool**: Vite

## 页面列表

1. **登录页** - 教师账号登录
2. **学生列表** - 查看所有学生，筛选/搜索
3. **测评详情** - 查看单个学生的完整报告
4. **生成链接** - 为学生生成入口 Token，为家长生成分享链接
